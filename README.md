# learndinosaurs

LearnDinosaurs is an interactive app that helps you explore the prehistoric world like a real paleontologist. Whether you're a curious high school student or an aspiring researcher, it gives you tools to study, analyze, and discover dinosaurs with the depth and rigor of a PhD.
